### 安装方法
- 在机器人根目录下执行以下命令之一：
```bash
#使用Gitee(国内服务器推荐)
git clone https://gitee.com/cunyx/xmz-plugin.git ./plugins/xmz-plugin
```
```bash
#使用GitHub(国外服务器推荐)
git clone https://github.com/cunyx-xiaomizhou/xmz-plugin.git ./plugins/xmz-plugin
```
- 安装必须的依赖
```bash
pnpm i
```
