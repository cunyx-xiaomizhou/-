# 填写前必看
## 简介
恭喜你，打开了正确的文件夹，超过了80%的插件使用者。下面是填写时的注意事项：
> 1. 你需要寻找需要修改功能对应的配置文件(文档末尾会给出指令对应的名称)
> 2. 打开并按照标准`json`语法修改文件
> 3. 请不要修改文件中的`version`字段，否则可能会使更改失效
> 4. 请不要修改本文件，可能会使更新冲突

## 单群自定义配置
假如你要单群自定义的群号为`123456`，则需要按照以下格式自定义：
```json
{
    "config": {
        "xxx_1": "xxxx",
        "xxx_2": "xxxx",
        "123456": {
            "xxx_2": "xxxx"
        }
    },
    "tip": {
        "xxx_1": "xxxx",
        "xxx_2": "xxxx"
    },
    "version": "xxxx"
}
```
说白了就是以`群号`为键把自定义内容放在`config`键下
